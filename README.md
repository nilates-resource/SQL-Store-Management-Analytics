# SQL-Store-Management-Analytics
Relational database design for a retail store including stock management, sales tracking, and profit-loss analytics using T-SQL.
# Retail Management System: Database Design & Business Analytics

This repository contains a comprehensive SQL-based database management system for a retail store ("Magaza"). The project demonstrates professional database architecture, business logic implementation, and performance optimization techniques.

## Key Features
*Relational Database Design:Includes 10+ interconnected tables (Products, Suppliers, Employees, Customers, Sales, Stocks, Expenses) with enforced integrity (Primary Keys, Foreign Keys, Check Constraints).
*Business Intelligence (Views): Custom views for "Top Selling Products," "Monthly Spendings," "Stock Entries," and "Customer Spending Habits."
*Advanced Automation (Stored Procedures): Custom procedures for CRUD operations (Update Product, Add Employee, Delete Customer) to ensure safe data manipulation.
*Performance Optimization: Non-clustered indexes on high-traffic columns (TCKN, Phone, City, Date) to speed up complex queries.
*Financial Analytics: Advanced SQL queries to calculate Daily Profit, Average Monthly Profit, and Loss/Profit analysis per product.

## Technical Stack
*Language: T-SQL (MS SQL Server)
*Concepts: Normalization, Views, Stored Procedures, Indexes, Aggregation Functions.

## Business Logic Focus
The system is designed to provide a 360-degree view of a retail business, bridging the gap between operational data and strategic decision-making. It calculates real-time margins and helps identify underperforming products.

# Perakende Mağaza Yönetim Sistemi: Veritabanı Tasarımı ve İş Analitiği

Bu proje, bir perakende mağazasının ("Magaza") tüm operasyonel süreçlerini yönetmek için tasarlanmış kapsamlı bir SQL veritabanı yönetim sistemidir. Profesyonel veritabanı mimarisi ve iş analitiği yetkinliklerini sergilemektedir.

## Öne Çıkan Özellikler
*İlişkisel Veritabanı Mimarisi: Ürünler, Tedarikçiler, Çalışanlar, Müşteriler, Satışlar, Stok ve Giderler gibi 10'dan fazla tablo arasında kurulan ilişkisel yapı.
*İş Zekası (Views): "En Çok Satan Ürünler", "Müşteri Harcamaları", "Stok Hareketleri" gibi stratejik raporlar için hazırlanmış görünümler.
*Gelişmiş Otomasyon (Stored Procedures): Güvenli veri yönetimi için hazırlanmış Ürün Güncelleme, Çalışan Ekleme ve Müşteri Silme prosedürleri.
*Performans Optimizasyonu: Sorgu hızını artırmak amacıyla kritik kolonlar (TCKN, Telefon, Şehir, Tarih) üzerinde kurgulanmış indeksler.
*Finansal Analiz: Günlük kâr, ortalama aylık kâr ve ürün bazlı kâr/zarar durumunu hesaplayan gelişmiş sorgular.

## Teknik Detaylar
*Dil: T-SQL (MS SQL Server)
*Kavramlar: Normalizasyon, View, Stored Procedure, Indexing, Agregasyon Fonksiyonları.
